    @function palette($palette, $tone: base) { ... }
    View source

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $palette

    Name of the color you want

    		String—none
    $tone

    (Optional) - Color Swatch

    		Stringbase

    Example

    Basic usage: use the default base color from the shared color palette

    .c-selector {
  color: palette(blue);
}

    Use a shade of a particular color

    .c-selector {
  color: palette(blue, darker);
}

    Description

    Sass Map of the shared color palette in Bolt.

    Type

    Map

    Map structure

    Map keyNameMap keyDescriptionMap keyTypeMap keyValue
    colors

    default namespace to store swatches. Allows for different groups of colors to be created. (ex. social media)

    		Map—none
    colors.color

    the name of the group of color values (ex. blue)

    		Map—none
    colors.color.base

    [required] -- default color used if a shade isn't specified.

    		Color—none
    colors.color.darker

    Example of storing a darker color shade
